PROJECT 01

Malbon Golf 말본골프

React Website (Redux / Router / TypeSctipt)

Point Color

미국에서 시작된 라이프스타일 브랜드 사이트 '말본골프(Malbon Golf)'입니다.

React를 활용하여 다양한 사용자에 맞게 반응할 수 있도록 반응형 웹페이지로,
Axios를 활용하여 외부 링크에 저장된 상품 데이터를 받아왔습니다.

새롭게 공개된 상품들과 항목별 인기 있는 상품들, 그리고 소비자들을 위한
코디 콘텐츠와 가장 인기 있는 상품들을 보다 쉽게 접할 수 있게 배치하였습니다.

TypeScript를 활용하여 JavaScript에서 일어날 수 있는 버그를 줄였으며,
상품 구매페이지인 Sub Page도 함께 제작하였습니다.

Point Color
